Smart-Academy-Manager (학원 통합 관리 솔루션)
소개
이 프로젝트는 개인 학원 및 소규모 교육 기관을 위해 설계된 학원 통합 관리 시스템입니다. 학생 데이터 관리의 복잡함을 해결하고, 효율적인 출결 및 상담 기록 자동화를 통해 교육 서비스의 품질을 높이는 것을 목표로 합니다.

주요 기능
학생 원스톱 관리: 학생 인적 사항, 등록 과목, 수강료 납부 현황을 한눈에 관리합니다.

간편 출결 시스템: 학생 입퇴실 시 자동으로 출결 상태를 기록하고, 학부모 알림 연동을 지원합니다.

상담 및 성적 히스토리: 학생별 상담 내용과 성적 변화를 체계적으로 기록하여 맞춤형 학습 가이드를 제공합니다.

데이터 분석 리포트: 월별 수강생 통계 및 성적 추이 데이터를 시각화하여 경영 효율을 높입니다.
