4. 핵심 문항 심층 분석

이번 시험에서 오답률이 가장 높고 등급을 가르는 결정적인 역할을 한 핵심 문항을 선정하여 상세히 분석합니다.

🔢 18번   로그 부등식 — 정수 x의 개수 조건 최상
💡 핵심 포인트

x²−x·log₃3n+log₃n≤0을 만족하는 정수 x의 개수가 정확히 3이 되도록 하는 자연수 n의 개수를 구하는 문제입니다. A=log₃n으로 치환 후 이차부등식의 근 사이 정수 개수를 분석해야 합니다.

🔎 왜 어려웠을까?

A=log₃n 치환 후 두 근 사이에 정수 x가 정확히 3개가 되는 A 범위를 설정하고, 이를 다시 n 범위로 역변환하는 이중 치환 과정이 복잡합니다.

📚 필요 개념
  • 이차부등식의 해 (두 근의 위치)
  • 로그를 이용한 치환 (A = log₃n)
  • 근과 계수의 관계 (근의 합·곱)
🚀 단계별 공략 솔루션
  1. A=log₃n으로 치환 → x²−x(A+1)+A≤0 변환
  2. 인수분해: (x−1)(x−A)≤0 → 두 근은 1과 A
  3. 두 근 사이 정수가 3개인 A의 범위 탐색
  4. A=log₃n의 범위를 n 범위로 역변환 후 자연수 n 개수 산출
📐 서술형3   지수·로그함수 합성 최대최솟값 상
💡 핵심 포인트

10≤x≤100 범위에서 f(x)=2^(log₂x)²−4×2^(log₁₀₀x)의 최대·최솟값을 구하는 문제입니다. 로그를 치환하여 이차 형태로 변환하는 핵심 아이디어를 찾는 것이 관건입니다.

🔎 왜 어려웠을까?

단순 대입으로는 풀 수 없고, t=log₂x로 치환하여 이차함수 형태로 바꾸는 과정과 x의 범위를 t의 범위로 변환하는 논리적 연결이 어렵습니다.

📚 필요 개념
  • 지수·로그함수 합성
  • 치환을 통한 이차함수 변환
  • 범위 변환 (x → t)
🚀 단계별 공략 솔루션
  1. t=log₂x 치환 → x:10~100이면 t 범위 설정
  2. f를 t에 대한 이차식으로 변환
  3. 구간 내 최대·최솟값 탐색 (꼭짓점·경계값 비교)
  4. M, m 도출 후 M+m 계산
🔺 서술형4   삼각함수 부등식 — θ의 범위 최상
💡 핵심 포인트

모든 실수 x에 대해 x²+2x·sinθ+1>0이 항상 성립하는 θ의 범위를 구하는 문제입니다. 이차부등식이 모든 실수에서 성립할 조건인 판별식 D<0을 이용합니다.

🔎 왜 어려웠을까?

이차부등식의 "항상 성립" 조건을 삼각함수와 연결하는 발상이 필요하며, D<0에서 sinθ의 범위, 단위원에서 θ의 범위로 변환하는 과정이 복잡합니다.

📚 필요 개념
  • 이차부등식의 항상 성립 조건 (D<0)
  • 삼각함수 부등식 |sinθ|<1
  • 삼각함수 범위 → θ 범위 변환
🚀 단계별 공략 솔루션
  1. 판별식 D=(2sinθ)²−4<0 설정
  2. 4sin²θ<4 → |sinθ|<1
  3. sinθ≠±1 → θ≠π/2, 3π/2 (0≤θ<2π)
  4. 최종: 0≤θ<π/2 또는 π/2<θ<3π/2 또는 3π/2<θ<2π
